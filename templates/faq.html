Welcome to FaQ about PublicPianos.org

Who is behind PublicPianos.org?

The original creator of the page is ElJugaZrc, but the page is uploaded to github so anyone can request any change. In summary, there is not a single person or a company, anyone can work in this page.

How can i submit a piano to the page?

Anyone can submit a public piano by joining the discord server. To submit a piano you will need to send a google maps link and a photo or video of the piano. Also, you should add a description of it so we can know the conditios of the piano. You will find more information in the discord server. We use discord so we can prove that the piano is real and to avoid ddos.

Why did I decide to make PublicPianos.org?

I decided to create PunlicPianos.org because I play the piano and sometimes I wander if there was a page in which I could see if there were a piano somewhere near me. I couldnt find one so i decided to create my own. Also, i wanted to learn coding so i made this as my first project.

Why do you use OpenStreetMap and not google maps?

We use OpenStreetMap cause is free. I dont have a lot of money so didnt wanted to risk and use google maps API. So i decided to use Leaflet and OpenStreetMap.

Why is the page so simple?

At the moment in which im making this website, i dont know a lot about coding and this is my first project, so im making preaty only the important parts.
